Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}

```

Ответ:
```
Программа выведет числа от 0 до 9, а затем программа уйдет в deadlock.
Так произойдет из-за того, что наш цикл по каналу будет ожидать новых значений из канала.
Но в канал уже ничего не будет отправляться, и он так и останется открытым. 
Цикл по каналу заканчивается только если канал закрыт.
```
